Где найти API-ключ?

{% if mp == 'wildberries' %}
  1. Зайдите в Личный кабинет WB → Настройки
  2. Раздел Доступ к API
  3. Нажмите Создать новый токен
  4. Выберите тип: Статистика
  5. Скопируйте токен и вставьте выше
{% elif mp == 'ozon' %}
  1. Зайдите в Личный кабинет Ozon Seller
  2. Настройки → API-ключи
  3. Нажмите Добавить ключ
  4. Скопируйте Client-ID и API-ключ
{% elif mp == 'yandex_market' %}
  1. Зайдите в partner.market.yandex.ru
  2. Настройки → API
  3. Создайте токен с доступом к статистике
  4. Campaign ID найдите в адресной строке кабинета
{% endif %}
