Как создать API-ключ в WB

  1. Зайдите в Личный кабинет WB → Настройки
  2. Раздел Доступ к API
  3. Нажмите Создать новый токен
  4. Придумайте название, например: CostBird
  5. Поставьте галочки напротив всех следующих разделов:
✓
Статистика
Продажи, заказы, возвраты, комиссии, логистика, штрафы
✓
Маркетплейс
Остатки на складах, поставки, заказы FBS
✓
Реклама
Расходы на рекламу, кампании, CTR, ROI
✓
Аналитика
ABC-анализ, воронка продаж, поисковые запросы
✓
Цены и скидки
Юнит-экономика, управление ценами
✓
Контент
Карточки товаров, фото, описания
  1. Нажмите Создать токен
  2. Скопируйте ключ — он показывается только один раз!
  3. Вставьте его в поле слева
💡 Минимум для старта: галочки Статистика и Маркетплейс. Остальные добавляют дополнительную аналитику.
{% elif mp == 'ozon' %}

Как создать API-ключ в Ozon

  1. Зайдите в Ozon Seller → Настройки
  2. Раздел API-ключи
  3. Нажмите Добавить ключ
  4. Выберите роль: Admin read only
  5. Скопируйте Client-ID (число) и API-ключ
💡 Роль Admin read only даёт доступ ко всем данным для чтения: продажи, финансы, аналитика, реклама.
{% elif mp == 'yandex_market' %}

Как создать токен в Яндекс Маркет

  1. Зайдите в partner.market.yandex.ru
  2. Нажмите на имя аккаунта → OAuth-токен
  3. Создайте токен с правами: Управление кампаниями и Статистика
  4. Campaign ID найдите в адресной строке кабинета:
    .../businesses/XXXXX/shops/CAMPAIGN_ID/...
💡 Один токен даёт доступ ко всей статистике — продажи, комиссии, логистика, реклама.
